Indian Railway Management System

This project is a simple prototype of a Railway Management System built using HTML and CSS. It allows passengers to search trains, view details, and fill out booking forms.

Our aim is to provide a smooth and easy-to-use ticket booking experience. Future versions can be extended with JavaScript, databases, and server-side technologies to make it a complete system.

Features:

  • Search Trains by Source, Destination, and Date
  • Passenger Booking Form
  • Simple, Clean UI with Responsive Design
